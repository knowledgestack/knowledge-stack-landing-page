# Blog Posts Directory

This directory contains all blog posts in Markdown format.

## How to Write a New Blog Post

1. Create a new `.md` file in this directory
2. Use the following frontmatter format at the top:

```markdown
---
title: "Your Blog Post Title"
date: "YYYY-MM-DD"
author: "Author Name"
excerpt: "A brief description that appears in the blog listing"
tags: ["Tag1", "Tag2", "Tag3"]
---
```

3. Write your content using Markdown syntax
4. The file will automatically appear in the blog listing

## Markdown Features Supported

- Headers (# ## ###)
- **Bold** and *italic* text
- Lists (ordered and unordered)
- Links: [text](url)
- Code blocks with syntax highlighting
- Images: ![alt](url)
- Tables
- Blockquotes

## Example Post Structure

```markdown
---
title: "My First Blog Post"
date: "2025-01-20"
author: "John Doe"
excerpt: "This is what readers will see in the blog listing"
tags: ["Tutorial", "Product"]
---

# My First Blog Post

Introduction paragraph here.

## Section 1

Content for section 1.

## Section 2

Content for section 2.

### Subsection

More detailed content.

## Conclusion

Wrap up your post here.
```
